Installation

  $ npm i gat -g

Adminstration

  $ gat -h

    Usage: gat [options] [command]

    Commands:

      config                 Show config
      start                  Start gat
      stop                   Stop gat
      empty                  Empty cache

    Options:

      -h, --help     output usage information
      -V, --version  output the version number

  $ gat start
    info: Starting gat on port 1947

  $ gat stop
    warn: Stopping gat

  $ gat config
    info: {
      "port": "1947",
      "pidFile": "gat.pid",
      "logFile": "gat.log"
    }

  $ gat empty
    warn: Emptying cache
