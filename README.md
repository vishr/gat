Cacher
======

An HTTP caching server

## Installation
```bash
  $ npm i gat -g
```

## Usage
```bash
  $ gat -h
  
    Usage: gat [options] [command]
  
    Commands:
  
      config                 Show config
      start                  Start gat
      stop                   Stop gat
      empty                  Empty cache
  
    Options:
  
      -h, --help     output usage information
      -V, --version  output the version number
```
